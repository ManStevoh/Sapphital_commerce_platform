## Purpose

Volume 15 defines SCP's **product and technology roadmap** from Nigeria GA through 2030 — mobile, POS, ERP, Academy integration, regional expansion, enterprise features, and long-term platform vision.

## Scope

- Roadmap horizons and capability map
- Mobile (React Native) and deep mobile architecture
- POS module and omnichannel retail
- ERP, CRM, and accounting integrations
- Sapphital Academy integration
- West Africa and global expansion (including GDPR readiness)
- Enterprise tier and features roadmap
- Platform vision 2030

## Out of Scope

- Sprint backlog
- Financial projections
- Implementation code (see Volume 21)
